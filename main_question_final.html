The list of scores can be organized by breaking it into small pieces and putting them back together in order, or by simply comparing neighbors and swapping them until everything is sorted. To find a specific score, you can either check each value one at a time from the start, or—after sorting—keep cutting the search area in half until you reach the score you want. These basic ideas make the overall process easy to understand for a beginner.
Problem Statement

A school maintains a list of students' test scores for analysis.
Given an array scores[] of integers representing the marks scored by students, you must perform the following operations:

• Sort the scores using Merge Sort
• Sort the scores using Bubble Sort (for comparison)
• Search for a particular student's score using Linear Search
• Search for a particular student's score using Binary Search (after sorting)

Your task is to write functions to perform each of these operations and use them to process the given data.

You have to find the optimal way to solve the problem based on insights from the previous steps.
